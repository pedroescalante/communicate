
# Communicate

This is a PHP Package that can be used to connect your PHP application to Twilio Call and SMS capabilities. 

## Getting Started

The package is oriented to be used in Laravel Framework. 

Add the package to your composer.json file.
```
	"require": {
		"pedroescalante/communicate": "dev-master"
	},  
```

## Running the tests

In the root folder of your project, run

```
vendor/bin/phpunit
```
